
### 1. Алгоритм FizzBuzz
Генерація послідовності елементів на основі кратності вхідного числа. 
- **Логіка:** Заміна числових значень на дескриптори "Fizz", "Buzz" або їх комбінацію.
- **Складність:** $O(n)$ за часом та $O(n)$ за пам'яттю, оскільки алгоритм потребує ітерації по всьому діапазону та збереження результатів у масиві.

### 2. Визначення знаку числа
Функція для математичного аналізу вхідного значення.
- **Особливість:** Забезпечує коректну обробку граничних значень, включаючи `Infinity`, `-Infinity` та `NaN`.
- **Складність:** $O(1)$ — виконання операції за константний час.

### 3. Перевірка високосного року
Аналітичний модуль для верифікації року за григоріанським календарем.
- **Метод:** Використання логічних операторів для перевірки кратність на 400 та 4/100.
- **Складність:** $O(1)$ — обчислення відбувається миттєво незалежно від значення року.

### 4. Класифікатор трикутників
Геометричний модуль для визначення типу фігури за трьома сторонами.
- **Валідація:** Попередня перевірка на нерівність трикутника для виключення некоректних даних.
- **Складність:** $O(1)$ — фіксована кількість логічних порівнянь.

### 5. Розрахунок академічного рейтингу 
Система конвертації числових балів у літерні еквіваленти за шкалою A-F з урахуванням модифікаторів.
- **Функціонал:** Динамічне додавання суфіксів `+` та `-` на основі залишку від ділення.
- **Складність:** $O(1)$ — алгоритм базується на арифметичних операціях та умовних переходах.
