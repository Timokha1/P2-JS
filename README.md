### 1. FizzBuzz Classic
- **Рішення**: Використано цикл `for` від 1 до `n`. На кожній ітерації число перевіряється на кратність за допомогою оператора `%`.

### 2. Number Sign
- **Рішення**: Пряма перевірка через `if/else` та обробка спеціальних типів даних (`NaN`).


### 3. Leap Year Checker
- **Рішення**: Реалізація календарного правила: рік має ділитися на 400 АБО бути кратним 4, але не кратним 100.
- **Складність**: Time $O(1)$, Space $O(1)$.

### 4. Triangle Type Classifier
- **Рішення**: Спочатку перевіряється теорема про нерівність сторін ($a+b > c$), потім порівнюються самі сторони.

### 5. Grade Letter Calculator
- **Рішення**: Комбінація базових умов для літери (A, B, C...) та математичного розрахунку залишку від ділення (`% 10`) для визначення знаку +/-.
